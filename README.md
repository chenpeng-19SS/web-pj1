## Bonus完成情况

### 更复杂的图片处理

将图片使用background来进行加载，同时设置centre和cover，实现了图片比例不协调的问题。

代码如下

```
.list-image1 {
    width: 250px;
    height: 250px;
    margin: 25px;
    border-radius: 20px;
    background: url(../../images/travel/index/1-1.jpg);
    background-position: center;
    background-size: cover;
}
```
